    Task Board

    JavaScript  ·  CSS Grid  ·  2024

    A Kanban task manager. Drag and drop that works with a keyboard too — which was the hard part. I followed the W3C ARIA drag-and-drop pattern and learned more about focus management than anything else I've done.

    What I focused on

    • Keyboard drag-and-drop using ARIA grabbed / dropeffect
    • Status announcements via aria-live="assertive"
    • localStorage persistence — tasks survive a page refresh
    • CSS Grid column layout, no frameworks
    • JavaScript
    • CSS Grid
    • localStorage
    • ARIA
